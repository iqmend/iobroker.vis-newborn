## Dokumentationsstruktur

Das offizielle ioBroker.docs-Repository ist wie folgt aufgebaut:

### Hauptordner
| Ordner | Zweck |
|--------|-------|
| `docs/` | Primäre Dokumentationsquellen (nach Sprache organisiert) |
| `admin/` | Admin-Adapter Konfiguration und Dokumentation |
| `engine/` | Build- und Webseiten-Generierungssystem (Node.js/Gulp) |
| `blog/` | Blog-Beiträge in mehreren Sprachen |
| `info/` | Metadaten, News, Info-Adapter-Dokumentation |
| `archive/` | Historische Dokumentation früherer Versionen |
| `media/` | Offizielle Branding-Assets und Logos |

### Sprachen
- **Deutsch (de)** - Hauptsprache
- **Englisch (en)** - Vollständig
- **Russisch (ru)** - Verfügbar
- **Chinesisch vereinfacht (zh-cn)** - Verfügbar
